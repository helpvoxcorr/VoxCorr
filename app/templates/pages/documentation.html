Introduction

VoxCorr est une application de correction audio de copies d'élèves. L'enseignant dicte sa correction oralement, l'IA (Mistral) la transcrit et structure automatiquement le retour écrit. Un QR code est généré pour chaque élève, donnant accès à sa correction sans compte requis.

Démarrage rapide : Créez une classe → ajoutez vos élèves → créez un devoir → enregistrez les corrections → publiez et partagez les QR codes.

Créer une classe

Depuis le tableau de bord, cliquez sur Nouvelle classe. Renseignez le nom de la classe, la discipline et l'année scolaire.

Cliquer sur Nouvelle classe dans le tableau de bord.
Renseigner le nom (ex : 3ème B), la discipline (ex : SVT) et l'année.
Valider. La classe apparaît dans la section correspondant à sa discipline.

Ajouter des élèves

Dans la page de la classe, utilisez le formulaire en bas de la liste pour ajouter un élève. Le prénom et le nom réel sont chiffrés (AES-256) en base de données. Un alias mnémotechnique est généré automatiquement (ex : Renard-07).

Le vrai nom n'est jamais affiché publiquement. Seul l'enseignant connecté peut le déchiffrer.

Modifier ou supprimer un élève

Les boutons et sont disponibles sur chaque ligne. La suppression d'un élève efface également toutes ses corrections.

Créer un devoir

Dans la page d'une classe, cliquez sur Nouveau devoir. Renseignez le titre, la date, le total de points, et ajoutez autant de questions que nécessaire avec leur barème.

Définissez bien vos questions avant de commencer les corrections — Mistral s'appuie sur leurs libellés pour extraire les notes automatiquement.

Enregistrer une correction

Sur la page du devoir, cliquez sur Corriger en face d'un élève. La page d'enregistrement s'ouvre.

Cliquer sur le bouton micro vert pour démarrer l'enregistrement.
Dicter la correction en mentionnant explicitement les notes : « Question 1, trois virgule cinq sur cinq… »
Cliquer sur le bouton stop rouge. L'audio est capturé, la transcription apparaît.
Cliquer sur Sauvegarder et analyser. Mistral génère la synthèse et extrait les notes.
Vérifier les notes dans le tableau à gauche, ajuster si besoin, puis Publier et générer QR.
Prononcez les notes clairement : « trois sur cinq », « deux virgule cinq sur quatre ». Évitez « trois et demi » qui peut être mal transcrit.

QR Code & page élève

Après publication, un QR code unique est généré pour chaque élève. L'élève scanne ce QR avec son téléphone et accède à sa correction — sans compte, sans application.

Déposer le QR code sur Pronote / Moodle / EcoleDirecte

Téléchargez le QR code en PNG depuis la page de correction (bouton ). Le fichier est nommé automatiquement classe_devoir_alias.png. Glissez-le dans votre espace numérique de travail habituel.

Consultation hors-ligne

La page de correction se met en cache automatiquement à la première visite. L'élève peut la relire sans connexion internet. Le fichier audio se télécharge à la demande via le bouton Télécharger pour écoute hors-ligne.

Imprimer les corrections

Sur la page du devoir, cliquez sur Imprimer corrections. Une page s'ouvre avec un cadre par élève (tri alphabétique), contenant la synthèse écrite et le QR code. Faites Ctrl+P ou cliquez Imprimer / Exporter PDF.

Dans les paramètres d'impression Chrome, activez Arrière-plans graphiques pour un rendu optimal. Choisissez A4 portrait.

Anonymisation RGPD

Les noms et prénoms des élèves sont chiffrés en base de données (algorithme Fernet, AES-128). Seul l'enseignant connecté peut les déchiffrer. Les exports publics (QR codes, page élève) n'affichent que l'alias anonyme.

Les logs de consultation (qui a vu quoi, quand) stockent l'adresse IP sous forme de hash SHA-256 irréversible — jamais l'IP brute.
